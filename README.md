# Python程序自动打包工具 - 基础版与增强版

## 项目简介

这是一套Python程序自动打包工具，包含基础版和增强版两个版本，专为将Python脚本一键打包成可执行文件（.exe）而设计。这两个程序都提供了简单直观的图形界面，让即使没有命令行经验的用户也能轻松完成打包工作。

**重要说明：这两个程序完全由AI编写，从代码逻辑到用户界面，都是人工智能生成的成果。**

## 版本对比

### 基础版（自动打包.py）

1.**核心功能**：基本的Python程序打包功能

2.**主要特点**：

  *简单直观的界面设计

  *支持打包成单个文件或目录结构

  *可选隐藏控制台窗口

  *自动检测并安装PyInstaller

  *后台线程处理避免界面卡顿

3.**适用场景**：适合简单的Python脚本打包需求，无需复杂配置

### 增强版（自动打包-增强版.py）

1.**核心功能**：在基础版上增加了高级功能和选项

2.**新增特性**：

  *完整的打包选项面板

  *MediaPipe库的特殊支持

  *可配置的隐藏导入项

  *额外命令行参数支持

  *实时打包日志显示

  *打包完成后的文件大小显示

  *一键打开输出目录功能

  *清理临时文件选项

3.**适用场景**：适合复杂的Python项目打包，特别是使用MediaPipe、OpenCV等库的计算机视觉项目

## 使用方法

### 基础版使用步骤

1.运行程序：python 自动打包.py

2..点击"浏览"按钮选择要打包的Python文件

3.根据需要选择打包选项：

  *打包成单个文件：所有依赖打包到一个exe中

  *隐藏控制台窗口：适合GUI程序，不显示命令行窗口

4.点击"开始打包"按钮

5.等待打包完成，程序会显示成功或错误信息

### 增强版使用步骤

1.运行程序：python 自动打包-增强版.py

2.点击"浏览"按钮选择要打包的Python文件

3.在打包选项区域配置：

  *单文件模式

  *控制台窗口显示/隐藏

  *MediaPipe支持（如果使用）

  *清理临时文件选项

4.在高级选项中配置：

  *需要隐藏导入的模块

  *额外的PyInstaller参数

5.点击"开始打包"按钮

6.在日志区域查看实时打包进度

7.打包完成后可选择打开输出目录

## 技术栈

两个版本都基于以下技术：

Python 3.x

Tkinter（GUI框架）

PyInstaller（打包引擎）

subprocess（系统命令调用）

threading（多线程处理）

增强版额外使用了：

tempfile（临时文件管理）

shutil（文件和目录操作）

## 注意事项

1.确保已安装Python环境

2.首次使用时会自动安装PyInstaller

3.打包过程中请勿关闭程序窗口

4.生成的exe文件位于源文件所在目录的dist子目录下

5.增强版的MediaPipe支持选项仅在使用MediaPipe库时才需要勾选
